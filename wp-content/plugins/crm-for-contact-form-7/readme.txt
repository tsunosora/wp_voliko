== Description ==

This CF7 Wordpress plugin sends form submissions from Contact Form 7 to Cloodo Worksuite to help capture lead, manage client workflow and then backup database for Contact Form 

### Feature of Contact Form 7 CRM plugin:

- By simply installing the plugin, it will automatically begin to capture form submissions from contact form 7.
- All your Contact Form 7 data will save on Lead management system. If your Contact Form 7 have fields name is your-name, it will be converted to customer name on Lead management. If it have field name is your-email or type is email will save to customer email. 
- All other fields on your contact form will save to lead note.No configuration is needed
- Save Contact Form 7 form submitted data to the database.
- Central database table for all contact form 7 forms
- Easy to use and lightweight plugin
- Developer friendly & easy to customize

== Frequently Asked Questions ==

= Is API required to run this plugin? =

Yes. Plugin will register an account automatic after you see first screen on erp.cloodo.com. All your data will be saved here. Your view will show on worksuite.cloodo.com.

= Why we build this plugin? =
Contact Form 7 is free contact form builder. This free Contact Form 7 CRM splugin adds entries management and all premium CRM features to free wordpress contact forms including contact form 7 so you will not have to install any other 3rd CF7 plugins 

= Do I have to pay for Cloodo Worksuite to use CRM? =

Cloodo Worksuite is free for any agency under 5 team member and you can use the CRM as well as other features for free life-time. You will need to upgrade to premium account if your organization member is over 5 team member while the Worksuite is really helpfull for your business.

= Do I have to create an account on Cloodo to use this plugin? =
When you install this CRM for Contact Form 7 plugin, the plugin itself will help to create an Workspace Account for your Wordpress admin smealessly and you just need to begin to work with your contact/ lead. 

= Can I set up Custom field on CRM to match with Contact Form 7 field? =

Yes, Worksuite CRM have advanced setting of Custom Field and you can set up unlimited Custom Field in your CRM system to matching with data lead form Contact Form 7. Please contact us to get instruction or request any custom development or API intergration at support@cloodo.comWill this plugin work if Contact Form 7 is not installed?
No, this plugin work only when Contact Form 7 is installed & activated.

= Does it need any configuration? =
No, it is a plug ‘n’ play plugin. Just install the plugin, and it will start capturing and displaying reports from supported form plugins.You can visit the settings page to see some basic config options.

= How can I view all contact form entries? =
You can view form entries from the main WordPress dashboard just by clicking on CF7 CRM menu

= Can I export contact to CSV or download our Contact? =
Yes you can do it with many other filter and other lead categorized management
